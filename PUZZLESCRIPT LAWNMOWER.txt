RULES
======

(Spawns random unmowed grass sprites)
[ UnmowedSpawn ] -> [ random Unmowed ]

(Only starts once X is pressed to start)
[ LEFT Player ] [ MowerNotStarted ] -> [ Player ] [ MowerLeftNotStarted ]
[ UP Player ] [ MowerNotStarted ] -> [ Player ] [ MowerUpNotStarted ]
[ RIGHT Player ] [ MowerNotStarted ] -> [ Player ] [ MowerRightNotStarted ]
[ DOWN Player ] [ MowerNotStarted ] -> [ Player ] [ MowerDownNotStarted ]

[ ACTION Player ] [ MowerLeftNotStarted ] -> [ Player ] [ MowerLeftStill ] SFX10
[ ACTION Player ] [ MowerUpNotStarted ] -> [ Player ] [ MowerUpStill ] SFX10
[ ACTION Player ] [ MowerRightNotStarted ] -> [ Player ] [ MowerRightStill ] SFX10
[ ACTION Player ] [ MowerDownNotStarted ] -> [ Player ] [ MowerDownStill ] SFX10



(((MOVEMENT ANIMATION)))
(Leftwards movement animation)
RIGHT [ MowerLeftLeft4 | MowerLeftRight4 ] -> [ MowerLeftTemp | ]
RIGHT [ MowerLeftLeft3 | MowerLeftRight3 ] -> [ MowerLeftLeft4 | MowerLeftRight4 ]
RIGHT [ MowerLeftLeft2 | MowerLeftRight2 ] -> [ MowerLeftLeft3 | MowerLeftRight3 ]
RIGHT [ MowerLeftLeft1 | MowerLeftRight1 ] -> [ MowerLeftLeft2 | MowerLeftRight2 ]
RIGHT [ No Solid | MowerLeftStill ] -> [ MowerLeftLeft1 | MowerLeftRight1 ]
[ MowerLeftTemp ] -> [ MowerLeftStill ]

(Upwards movement animation)
DOWN [ MowerUpUp4 | MowerUpDown4 ] -> [ MowerUpTemp | ]
DOWN [ MowerUpUp3 | MowerUpDown3 ] -> [ MowerUpUp4 | MowerUpDown4 ]
DOWN [ MowerUpUp2 | MowerUpDown2 ] -> [ MowerUpUp3 | MowerUpDown3 ]
DOWN [ MowerUpUp1 | MowerUpDown1 ] -> [ MowerUpUp2 | MowerUpDown2 ]
DOWN [ No Solid | MowerUpStill ] -> [ MowerUpUp1 | MowerUpDown1 ]
[ MowerUpTemp ] -> [ MowerUpStill ]

(Rightwards movement animation)
RIGHT [ MowerRightLeft4 | MowerRightRight4 ] -> [ | MowerRightTemp ]
RIGHT [ MowerRightLeft3 | MowerRightRight3 ] -> [ MowerRightLeft4 | MowerRightRight4 ]
RIGHT [ MowerRightLeft2 | MowerRightRight2 ] -> [ MowerRightLeft3 | MowerRightRight3 ]
RIGHT [ MowerRightLeft1 | MowerRightRight1 ] -> [ MowerRightLeft2 | MowerRightRight2 ]
RIGHT [ MowerRightStill | No Solid ] -> [ MowerRightLeft1 | MowerRightRight1 ]
[ MowerRightTemp ] -> [ MowerRightStill ]

(Downwards movement animation)
DOWN [ MowerDownUp4 | MowerDownDown4 ] -> [ | MowerDownTemp ]
DOWN [ MowerDownUp3 | MowerDownDown3 ] -> [ MowerDownUp4 | MowerDownDown4 ]
DOWN [ MowerDownUp2 | MowerDownDown2 ] -> [ MowerDownUp3 | MowerDownDown3 ]
DOWN [ MowerDownUp1 | MowerDownDown1 ] -> [ MowerDownUp2 | MowerDownDown2 ]
DOWN [ MowerDownStill | No Solid ] -> [ MowerDownUp1 | MowerDownDown1 ]
[ MowerDownTemp ] -> [ MowerDownStill ]



(((MOVEMENT INPUTS)))
(Detect input)
[ LEFT Player ] [ Buffer ] -> [ Player ] [ BufferedLeft ]
[ UP Player ] [ Buffer ] -> [ Player ] [ BufferedUp ]
[ RIGHT Player ] [ Buffer ] -> [ Player ] [ BufferedRight ]
[ DOWN Player ] [ Buffer ] -> [ Player ] [ BufferedDown ]

(Allows inputs to be made one frame late, game feels more responsive this way)
late [ NonEmptyBuffer ] [ MowerFrame1Front | MowerFrame1Back ] -> [ NonEmptyBuffer ] [ | MowerLeftStill ]

(Evaluates the input once you reach the next tile)
late [ BufferedLeft ] [ StillMower ] -> [ EmptyBuffer ] [ MowerLeftStill ]
late [ BufferedUp ] [ StillMower ] -> [ EmptyBuffer ] [ MowerUpStill ]
late [ BufferedRight ] [ StillMower ] -> [ EmptyBuffer ] [ MowerRightStill ]
late [ BufferedDown ] [ StillMower ] -> [ EmptyBuffer ] [ MowerDownStill ]

(Allows you to turn backwards on any frame of the animation (i.e. you don't have to wait until the next tile if you're turning backwards))
late [ BufferedLeft ] [ MowerRightRight2 | MowerRightLeft2 ] -> [ EmptyBuffer ] [ MowerLeftRight3 No Unmowed | MowerLeftLeft3 ]
late [ BufferedLeft ] [ MowerRightRight3 | MowerRightLeft3 ] -> [ EmptyBuffer ] [ MowerLeftRight2 No Unmowed | MowerLeftLeft2 ]
late [ BufferedLeft ] [ MowerRightRight4 | MowerRightLeft4 ] -> [ EmptyBuffer ] [ MowerLeftRight1 No Unmowed | MowerLeftLeft1 ]

late [ BufferedUp ] [ MowerDownDown2 | MowerDownUp2 ] -> [ EmptyBuffer ] [ MowerUpDown3 No Unmowed | MowerUpUp3 ]
late [ BufferedUp ] [ MowerDownDown3 | MowerDownUp3 ] -> [ EmptyBuffer ] [ MowerUpDown2 No Unmowed | MowerUpUp2 ]
late [ BufferedUp ] [ MowerDownDown4 | MowerDownUp4 ] -> [ EmptyBuffer ] [ MowerUpDown1 No Unmowed | MowerUpUp1 ]

late [ BufferedRight ] [ MowerLeftLeft2 | MowerLeftRight2 ] -> [ EmptyBuffer ] [ MowerRightLeft3 No Unmowed | MowerRightRight3 ]
late [ BufferedRight ] [ MowerLeftLeft3 | MowerLeftRight3 ] -> [ EmptyBuffer ] [ MowerRightLeft2 No Unmowed | MowerRightRight2 ]
late [ BufferedRight ] [ MowerLeftLeft4 | MowerLeftRight4 ] -> [ EmptyBuffer ] [ MowerRightLeft1 No Unmowed | MowerRightRight1 ]

late [ BufferedDown ] [ MowerUpUp2 | MowerUpDown2 ] -> [ EmptyBuffer ] [ MowerDownUp3 No Unmowed | MowerDownDown3 ]
late [ BufferedDown ] [ MowerUpUp3 | MowerUpDown3 ] -> [ EmptyBuffer ] [ MowerDownUp2 No Unmowed | MowerDownDown2 ]
late [ BufferedDown ] [ MowerUpUp4 | MowerUpDown4 ] -> [ EmptyBuffer ] [ MowerDownUp1 No Unmowed | MowerDownDown1 ]



(((USING DASH POWER)))
(Place the dashtemp)
[ ACTION Player ] [ DashPowerIcon ] -> [ Player ] [ DashPowerIcon BufferedDash ]

(Unless a directional input is buffered, place the temp dash.
If a directional input is buffered, this will wait until that input makes the mower turn.)
[ EmptyBuffer ] [ BufferedDash ] [ MowerMovingLeftFront ] -> [ EmptyBuffer ] [ random DashSound ] [ MowerMovingLeftFront DashLeftTemp ] SFX0
[ EmptyBuffer ] [ BufferedDash ] [ MowerMovingUpFront ] -> [ EmptyBuffer ] [ random DashSound ] [ MowerMovingUpFront DashUpTemp ] SFX0
[ EmptyBuffer ] [ BufferedDash ] [ MowerMovingRightFront ] -> [ EmptyBuffer ] [ random DashSound ] [ MowerMovingRightFront DashRightTemp ] SFX0
[ EmptyBuffer ] [ BufferedDash ] [ MowerMovingDownFront ] -> [ EmptyBuffer ] [ random DashSound ] [ MowerMovingDownFront DashDownTemp ] SFX0

[ DashSound1 ] -> [ ] SFX1
[ DashSound2 ] -> [ ] SFX2
[ DashSound3 ] -> [ ] SFX3
[ DashSound4 ] -> [ ] SFX4
[ DashSound5 ] -> [ ] SFX5

(Play the dash trail animations)
[ DashTrailHoriz4 ] -> [ ]
[ DashTrailHoriz3 ] -> [ DashTrailHoriz4 ]
[ DashTrailHoriz2 ] -> [ DashTrailHoriz3 ]
[ DashTrailHoriz1 ] -> [ DashTrailHoriz2 ]

[ DashTrailVert4 ] -> [ ]
[ DashTrailVert3 ] -> [ DashTrailVert4 ]
[ DashTrailVert2 ] -> [ DashTrailVert3 ]
[ DashTrailVert1 ] -> [ DashTrailVert2 ]

(Move the dash temp until it hits a solid or a carve pickup)
(Note: In regards to the CarvePowerPickup detection, it's done one tile later for vertical dashes due to the way CarveDashVertTemp works)
RIGHT [ no Solid no CarvePowerPickup | DashLeftTemp ] -> [ DashLeftTemp | DashTrailHoriz1 ]
DOWN [ no Solid | DashUpTemp no CarvePowerPickup ] -> [ DashUpTemp | DashTrailVert1 ]
RIGHT [ DashRightTemp | no Solid no CarvePowerPickup ] -> [ DashTrailHoriz1 | DashRightTemp ]
DOWN [ DashDownTemp no CarvePowerPickup | no Solid ] -> [ DashTrailVert1 | DashDownTemp ]

(If it hit a carve pickup, move until a solid is reached, while placing down carving temps)
RIGHT [ no Solid | DashLeftTemp ] -> [ DashLeftTemp | DashTrailHoriz1 CarveTemp ]
DOWN [ no Solid | DashUpTemp ] -> [ DashUpTemp | DashTrailVert1 CarveDashVertTemp ]
RIGHT [ DashRightTemp | no Solid ] -> [ DashTrailHoriz1 CarveTemp | DashRightTemp ]
DOWN [ DashDownTemp | no Solid ] -> [ DashTrailVert1 CarveDashVertTemp | DashDownTemp ]

(Place CarveTemps on both sides of the CarveDashVertTemp)
HORIZONTAL [ CarveDashVertTemp | ] -> [ CarveDashVertTemp | CarveTemp ]
[ CarveDashVertTemp ] -> [ ]

(Places carvetemps in a way such that the carve and dash powers synergize)
RIGHT [ CarvePowerIcon ] [ DashTrailHoriz1 ] -> [ CarvePowerIcon ] [ DashTrailHoriz1 CarveTemp ]
RIGHT [ CarvePowerIcon ] [ no CarveTemp | DashTrailVert1 | no CarveTemp ] -> [ CarvePowerIcon ] [ CarveTemp | DashTrailVert1 | CarveTemp ]

(Delete all mowers)
[ DashTemp ] [ MowerMoving ] -> [ DashTemp ] [ ]

(Mow lawn and pick up powerups where the dash trail touches)
[ DashTrail Unmowed ] -> [ DashTrail Mowed ]
